# Asteroids

## How to Run

**Unity Version:** Unity 6.4 (2D URP)

**Steps:**
1. Clone the repository
2. Open the project in Unity Hub
3. Open `Assets/_Asteroids/Scenes/GameScene.unity`
4. Press Play

**Controls:**
- `W` / `Up Arrow` — Thrust
- `A` / `Left Arrow` — Rotate left
- `D` / `Right Arrow` — Rotate right
- `Space` — Shoot
